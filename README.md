# Hotel Booking Website (Django)

A full-stack hotel booking web application built using Django.

## Features
- Hotel & flight booking
- Booking management
- Payments integration
- PDF invoice generation
- Modular Django apps

## Tech Stack
- Backend: Django (Python)
- Database: SQLite / MySQL (configurable)
- Frontend: Django Templates, HTML, CSS
- Payments: Razorpay

## Project Structure
- booking/
- flight/
- payment/
- signup/
- website/
